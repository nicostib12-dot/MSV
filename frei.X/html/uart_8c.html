Detailed Description

Driver UART/EUSART por hardware para el PIC18F4550.

Este archivo implementa las funciones necesarias para enviar información serial desde el PIC18F4550 hacia otro dispositivo, como un Arduino, un conversor USB-TTL o un computador.

En este proyecto, la UART se usa para enviar:

  • Mensajes de inicio del sistema.
  • Mensajes de error de sensores.
  • Estado del perfil seleccionado.
  • Temperatura medida por el DS18B20.
  • Ritmo cardíaco calculado a partir del MAX30102.
  • Estado de alarma del sistema.

Pines UART del PIC18F4550:

  • RC6: TX, transmisión de datos.
  • RC7: RX, recepción de datos.

Configuración usada:

  • Modo asíncrono.
  • Velocidad: 9600 baudios.
  • Formato típico: 8 bits de datos, sin paridad y 1 bit de parada.

void UART_Init ( void )

Inicializa el módulo EUSART del PIC18F4550.

Configura los pines RC6 y RC7 para el uso del módulo UART por hardware. También establece la velocidad de comunicación en 9600 baudios usando el oscilador interno de 8 MHz.

Configuración:

  • RC6 como TX.
  • RC7 como RX.
  • Modo asíncrono.
  • Alta velocidad.
  • Receptor continuo habilitado.
  • Transmisor habilitado.

Fórmula usada para el baud rate:

SPBRG = Fosc / (16 * Baud) - 1
SPBRG = 8000000 / (16 * 9600) - 1
SPBRG = 51
Note
Si se cambia la frecuencia del oscilador o la velocidad deseada, también debe modificarse el valor de SPBRG.

Configura la UART por hardware para trabajar en modo asíncrono a 9600 baudios. También habilita el transmisor y el receptor continuo.

Esta función debe llamarse antes de enviar cualquier dato por UART.

Note
La configuración interna de velocidad se encuentra en uart.c, dentro de la función UART_Init().

void UART_SendVitals ( unsigned char temp_int,
unsigned char temp_dec,
unsigned char bpm,

Envía los signos vitales y el estado del sistema por UART.

Envía una trama con los signos vitales del sistema.

Esta función genera una trama de texto con la información principal del sistema Cardio.X. El formato enviado permite observar los datos en el monitor serial de Arduino, un terminal serial o un conversor USB-TTL.

Formato enviado:

T:36.7 B:075 P:NOR AL:0

Donde:

  • T: temperatura en grados Celsius.
  • B: ritmo cardíaco en BPM.
  • P: perfil seleccionado.
    • NOR: perfil normal.
    • DEP: perfil deportista.
  • AL: estado de alarma.
    • 0: sin alarma.
    • 1: con alarma.
Parameters
temp_intParte entera de la temperatura.
temp_decParte decimal de la temperatura.
bpmRitmo cardíaco en BPM.
perfilPerfil seleccionado.
  • 0: perfil normal.
  • 1: perfil deportista.
alarmaEstado de alarma.
  • 0: no hay alarma.
  • 1: hay alarma.

Envía los signos vitales y el estado del sistema por UART.

void UART_WriteString ( const char * str)

Envía una cadena de caracteres por UART.

Envía una cadena de texto por UART.

La función recorre la cadena hasta encontrar el carácter nulo '\0'. Cada carácter se envía mediante UART_WriteChar().

Parameters
strPuntero a la cadena de texto que se desea enviar.

Envía una cadena de caracteres por UART.

Recorre la cadena recibida y envía cada carácter usando UART_WriteChar(). La cadena debe terminar con el carácter nulo '\0'.

Parameters
strCadena de caracteres que se desea transmitir.
Note
Para que el mensaje aparezca en una nueva línea en el monitor serial, se recomienda terminar la cadena con "\r\n".
