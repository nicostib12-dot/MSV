Detailed Description

Archivo de cabecera para el manejo de pantalla OLED SSD1306 128x64 por I2C.

Este archivo declara las funciones utilizadas para controlar una pantalla OLED basada en el controlador SSD1306. La comunicación se realiza mediante I2C por software usando los pines RB0 y RB1 del PIC18F4550.

La pantalla OLED se usa en el proyecto Cardio.X para mostrar:

  • Pantalla de inicio.
  • Temperatura medida por el DS18B20.
  • Ritmo cardíaco calculado con el MAX30102.
  • Perfil seleccionado.
  • Estado de alarma.
  • Estado de pausa.
  • Mensajes de error.

Características principales:

  • Resolución: 128 x 64 píxeles.
  • Organización interna: 8 páginas de 128 columnas.
  • Cada byte representa 8 píxeles verticales.
  • Dirección I2C típica del SSD1306: 0x3C.
  • Trama I2C desplazada usada en el código: 0x78.
Note
Este módulo depende del driver I2C definido en i2c.h.

◆ OLED_Clear()

void OLED_Clear ( void )

Limpia completamente la pantalla OLED.

Borra toda la pantalla llenandola con 0x00.

Esta función borra todas las páginas y columnas de la pantalla, dejando la memoria gráfica en cero.

Limpia completamente la pantalla OLED.

Estrategia: una transaccion I2C por pagina con 128 bytes cada una. Es mas eficiente que enviar byte a byte porque reduce el overhead de START/STOP/ACK.

Tiempo aproximado a 50 kHz: ~8 x (128 x 9 bits) / 50000 = ~185 ms. Nota: es lento por el bit-banging a 50 kHz. Para refrescar lineas individuales usa OLED_ClearLine().

◆ OLED_ClearLine()

void OLED_ClearLine ( unsigned char linea)

Limpia una línea o página específica de la pantalla OLED.

Borra solo una pagina (fila de 8px de alto).

La pantalla SSD1306 está organizada en 8 páginas. Cada página corresponde a una franja horizontal de 8 píxeles de alto.

Parameters
lineaPágina o línea que se desea limpiar. Valores típicos: 0 a 7.

Limpia una línea o página específica de la pantalla OLED.

Usar en el ciclo de refresco para actualizar solo las lineas que cambian, evitando el parpadeo de OLED_Clear().

Ejemplo: para actualizar la linea de temperatura: OLED_ClearLine(2); OLED_SetCursor(0, 2); OLED_WriteString("TEMP: 37.1");

Parameters
lineaNumero de pagina: 0 (arriba) a 7 (abajo).

void OLED_Command ( unsigned char cmd)

Envía un comando al controlador SSD1306.

Envia un byte de COMANDO al SSD1306.

Los comandos permiten configurar el comportamiento de la pantalla, como encendido, modo de direccionamiento, contraste, posición de memoria y otros parámetros internos del controlador.

Parameters
cmdComando que se enviará al SSD1306.

Envía un comando al controlador SSD1306.

El byte de control 0x00 le indica al chip que lo siguiente es un comando de configuracion (no datos de pixeles).

Trama completa: [START][0x78][0x00][cmd][STOP]

◆ OLED_Data()

void OLED_Data ( unsigned char data)

Envía un byte de datos gráficos a la pantalla OLED.

Envia un byte de DATOS (pixeles) al SSD1306.

Cada byte enviado representa una columna de 8 píxeles verticales dentro de una página de memoria del SSD1306.

Parameters
dataByte de datos que se escribirá en la memoria gráfica de la OLED.

Envía un byte de datos gráficos a la pantalla OLED.

El byte de control 0x40 indica que lo siguiente son datos para la memoria GDDRAM (pixeles de la pantalla). El puntero de columna avanza automaticamente tras cada byte.

Trama completa: [START][0x78][0x40][data][STOP]

Ubica el cursor en una columna y página específica.

Posiciona el cursor para la siguiente escritura.

Antes de escribir texto o datos, se debe ubicar el cursor en la posición deseada de la pantalla.

Parameters

◆ OLED_WriteChar()

void OLED_WriteChar ( char c)

Escribe un carácter en la posición actual de la OLED.

Dibuja un caracter en la posicion actual del cursor.

La función utiliza una fuente básica de 5x7 píxeles. Cada carácter ocupa normalmente 6 columnas: 5 columnas de dibujo y 1 columna de separación.

Parameters
cCarácter que se desea escribir en pantalla.
Note
La fuente incluida soporta principalmente espacio, números, letras mayúsculas y algunos símbolos básicos.

Escribe un carácter en la posición actual de la OLED.

Proceso:

  1. Determinar el indice del caracter en font5x7[].
  2. Enviar los 5 bytes del bitmap en una transaccion I2C.
  3. Enviar 1 byte 0x00 como columna de separacion.

El cursor avanza automaticamente 6 columnas (5+1).

Caracteres soportados: ' ' 0-9 A-Z : % . Cualquier otro caracter se muestra como espacio en blanco.

Parameters
cCaracter ASCII a dibujar.

Escribe una cadena de texto desde la posicion actual.

Esta función escribe carácter por carácter hasta encontrar el final de la cadena. Internamente usa OLED_WriteChar().

Parameters

Escribe una cadena de texto en la OLED.

Llama a OLED_WriteChar() para cada caracter hasta '\0'. El cursor avanza automaticamente.

Parameters
