Detailed Description

Driver básico para el sensor MAX30102 usando comunicación I2C.

Este archivo implementa las funciones necesarias para comunicarse con el sensor MAX30102 desde el PIC18F4550. El sensor se comunica mediante el bus I2C y permite obtener muestras de luz roja e infrarroja, las cuales luego son procesadas para estimar el ritmo cardíaco.

Funciones principales:

  • Escritura de registros internos del MAX30102.
  • Lectura de registros internos del MAX30102.
  • Verificación del identificador del sensor.
  • Configuración inicial del sensor.
  • Consulta de muestras disponibles en la FIFO.
  • Lectura de muestras RED e IR desde la FIFO.

El MAX30102 se comunica por I2C con:

  • SCL: línea de reloj.
  • SDA: línea de datos.

Verifica si el sensor MAX30102 está conectado correctamente.

Verifica la presencia del sensor MAX30102.

La función lee el registro PART ID del sensor. Para el MAX30102, el valor esperado del identificador es 0x15. Si el valor leído coincide, se considera que el sensor está presente y responde correctamente en el bus I2C.

Returns
1 si el sensor responde y el PART ID es correcto.
0 si no responde o el identificador no coincide.

Verifica si el sensor MAX30102 está conectado correctamente.

Lee el registro PART ID y compara su valor con el identificador esperado del sensor.

Returns
1 si el MAX30102 fue detectado correctamente.
0 si el sensor no responde o el identificador no coincide.

Inicializa el sensor MAX30102.

Esta función verifica primero la presencia del sensor mediante el registro PART ID. Si el sensor responde correctamente, se configuran sus registros principales para operar en modo SpO2, usando los LED rojo e infrarrojo.

Configuraciones realizadas:

  • Modo de operación SpO2.
  • Limpieza de registros de interrupción.
Returns
1 si el sensor fue inicializado correctamente.
0 si el sensor no fue detectado.

Inicializa y configura el sensor MAX30102.

Configura el sensor para trabajar en modo SpO2, limpia la FIFO, ajusta la corriente de los LED rojo e infrarrojo y deja listo el sensor para entregar muestras.

Returns
1 si la inicialización fue exitosa.
0 si el sensor no fue detectado.

◆ MAX30102_ReadSample()

unsigned char MAX30102_ReadSample ( unsigned long * red,
unsigned long * ir )

Lee una muestra completa RED e IR desde la FIFO del MAX30102.

Lee una muestra RED e IR desde la FIFO del MAX30102.

Cada muestra en modo SpO2 está compuesta por 6 bytes:

  • 3 bytes para el LED rojo.
  • 3 bytes para el LED infrarrojo.

El MAX30102 entrega datos de 18 bits, por lo que se toman los 2 bits menos significativos del primer byte y se combinan con los otros dos bytes.

Secuencia de lectura:

  • Verificar que existan muestras disponibles.
  • Apuntar al registro FIFO_DATA.
  • Leer 6 bytes por I2C.
  • Reconstruir el valor RED de 18 bits.
  • Reconstruir el valor IR de 18 bits.
Parameters
redPuntero donde se almacenará la muestra del LED rojo.
irPuntero donde se almacenará la muestra del LED infrarrojo.
Returns
1 si se leyó una muestra correctamente.
0 si no había muestras disponibles en la FIFO.

Lee una muestra completa RED e IR desde la FIFO del MAX30102.

Cada lectura completa en modo SpO2 está formada por 6 bytes: tres bytes para la seńal roja y tres bytes para la seńal infrarroja. La función reconstruye ambos valores como datos de 18 bits.

Parameters
redPuntero donde se almacenará la muestra del LED rojo.
irPuntero donde se almacenará la muestra del LED infrarrojo.
Returns
1 si se leyó una muestra correctamente.
0 si no había muestras disponibles.

◆ MAX30102_SamplesAvailable()

unsigned char MAX30102_SamplesAvailable ( void )

Consulta cuántas muestras hay disponibles en la FIFO del MAX30102.

Consulta cuántas muestras hay disponibles en la FIFO.

El MAX30102 almacena las muestras en una memoria FIFO interna. Esta función lee el puntero de escritura y el puntero de lectura para calcular cuántas muestras están pendientes por leer.

Returns
Número de muestras disponibles en la FIFO.
Note
La operación se limita con 0x1F porque la FIFO del MAX30102 trabaja con punteros de 5 bits.

Consulta cuántas muestras hay disponibles en la FIFO del MAX30102.

Compara el puntero de escritura y el puntero de lectura de la FIFO para calcular cuántas muestras están pendientes por leer.

Returns
Número de muestras disponibles en la FIFO del MAX30102.
